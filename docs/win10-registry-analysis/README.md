# Windows 10 Registry Deep Analysis for POLER-OS

Source: Windows 10 22H2 English x64 (Build 19045.2965)

## Directory Structure

- `json-results/` — Parsed JSON data from registry hives
- `reg-exports/` — Exported .reg files + DLL/function lists  
- `scripts/` — Python analysis scripts (reproducible)
- `../POLER-OS_Win10_Registry_Analysis.pdf` — Summary report

## NOT in offline hives (requires installed Windows)

- **ApiSetMap** — Generated by kernel from apisetchema.dll at boot
- **AppCertDLLs** — Empty in clean OS, created by anti-cheat software
- **AppCompat** — Empty in clean OS, populated by application compatibility layer
- **Diagnostics** — Empty in clean OS
- **Running services state** — Only ControlSet001/002 in offline image

To get these, install Windows in a VM and export from a live system.
